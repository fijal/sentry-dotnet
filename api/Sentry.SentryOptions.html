    Release

    The release version of the application.

    Declaration
    public string Release { get; set; }
    Property Value
    Type Description
    String
    Remarks

    This value will generally be something along the lines of the git SHA for the given project. If not explicitly defined via configuration or environment variable (SENTRY_RELEASE). It will attempt o read it from: AssemblyInformationalVersionAttribute

    Don't rely on discovery if your release is: '1.0.0' or '0.0.0'. Since those are default values for new projects, they are not considered valid by the discovery process.

    Examples

    721e41770371db95eee98ca2707686226b993eda 14.1.16.32451

    See Also
    https://docs.sentry.io/learn/releases/
